[Yuzu→Lu] 利用規約

本規約（以下「本規約」といいます）は、[Yuzu→Lu]（以下「当サービス」といいます）を利用するすべてのユーザー（以下「ユーザー」といいます）に適用されます。ユーザーは、当サービスを利用することによって、本規約に同意したものとみなされます。

第1条（定義）

  1. 当サービス

    2. 当サービスは、ユーザーが商品・サービスを貸し借り、または譲渡するためのプラットフォームを提供します。取引は、いかなる商品の「販売」ではなく、すべて「貸し借り」または「譲渡」に限られます。

  2. ユーザー

    3. 当サービスに登録し、利用する個人または法人を指します。

  3. 商品

    4. ユーザーが当サービス内で貸し借り、譲渡する物品やサービスを指します。

  4. マッチング

    5. ユーザーが商品やサービスの提供者と受け手を見つけるプロセスを指します。ユーザーは、このマッチングを通じて取引相手を見つけ、貸し借りや譲渡の契約を結びます。

第2条（サービス内容）

  1. 貸し借りおよび譲渡機能

    2. 当サービスは、ユーザー間での商品・サービスの「貸し借り」および「譲渡」の取引を提供します。ユーザーは、商品の所有権を移転せずに、一定の期間、または条件に基づいて他のユーザーに商品を貸し出したり、譲渡したりすることができます。

  2. マッチング機能

    3. ユーザーは、貸し借りや譲渡を希望する商品を投稿し、受け手を探すための検索、フィルタリング、メッセージ送信などの機能を利用できます。マッチングが成立すると、貸し借りや譲渡の条件について合意し、取引が開始されます。

  3. 評価・レビュー機能

    4. 取引終了後、ユーザーは取引相手に対して評価を行い、取引の信頼性や品質を評価することができます。

第3条（ユーザーの義務）

  1. 登録情報の提供

    2. ユーザーは、正確で最新の情報を登録し、常にその情報を更新する責任があります。不正確な情報や虚偽の情報を提供した場合、アカウント停止等の措置を取ることがあります。

  2. 取引の誠実性

    3. ユーザーは、商品の説明や状態について正確かつ誠実に記載する義務があります。また、取引が成立した場合、ユーザーは合意した内容に基づいて責任を持って取引を行わなければなりません。

  3. 禁止行為

    4. ユーザーは、以下の行為を行ってはならないものとします。

    1. 商品の不正な貸し借りまたは譲渡（例えば、盗品や違法な商品）
    2. 他のユーザーを欺く行為（虚偽の情報を提供する行為）
    3. 他のユーザーの権利を侵害する行為
    4. 商品やサービスに関して詐欺的な行為を行うこと
    5. サービス内での迷惑行為やスパム行為
    6. 寝ながら当サービスを操作する行為
    7. 当社スタッフへのサプライズプレゼントの送りつけ
    8. 当サービスの画面をスクショしSNSに「サービスすごすぎ！」と投稿し、当サービスに予期せぬバズりをもたらす行為
    9. 当サービスが突然フリーズした場合、怒りのあまりスマホを振り回す行為

第4条（友情の破壊）

  1. 紹介

    2. ユーザーが友人等に当サービスを紹介する際は、少なくとも10％の誇張を含めることとします。その結果、友情が深まったり壊れたりしても一切の責任を負いません。また、当サービスを友人に見せた結果、友情が破壊される場合があります。その際は「お互いの価値観が違ったんだな」と前向きに受け止めてください。

  2. 周囲への影響

    3. 友人と一緒に笑いすぎて場の空気が崩壊する場合、そこに責任は感じなくても構いません。

第5条（貸し借り・譲渡契約）

  1. 契約の成立

    2. ユーザー間で貸し借りまたは譲渡の条件に合意した時点で、当該取引は成立したとみなされます。契約内容は、商品やサービスの貸し借り期間、譲渡条件などに基づきます。

  2. 貸し借りの期間および譲渡条件

    3. 商品が「貸し借り」である場合、貸出期間や返却日を明確に設定し、双方がその期間を守る責任があります。譲渡の場合は、所有権が完全に移転することになります。

  3. 返品・返却

    4. 商品が貸し借りの場合、返却が必要となります。ユーザーは、契約に基づき期限内に商品の返却を行う義務を負います。返却に関する問題が発生した場合、ユーザー間で解決するものとします。

  4. 取引のキャンセル

    5. 取引開始後にキャンセルが発生する場合、ユーザーは速やかに相手に通知し、合意を得るものとします。キャンセルに伴う費用負担やその他の調整は、ユーザー間で協議して決定します。

第6条（ユーザーアカウントの管理）

  1. アカウントの管理責任

    2. ユーザーは、自身のアカウント情報を適切に管理し、第三者に不正利用されないようにする責任があります。

  2. アカウント停止・削除

    3. 当サービスは、以下の場合にユーザーのアカウントを停止または削除することがあります。

    1. 本規約に違反した場合
    2. 不正な行為が確認された場合
    3. 当サービスの運営に支障をきたす行為があった場合

第7条（料金・支払い）

  1. 手数料

    2. 当サービスは、ユーザー間の貸し借りや譲渡に対して、一定の手数料を徴収することがあります。手数料の詳細については、別途案内される場合があります。

  2. 支払い方法

    3. ユーザーは、当サービスが指定する支払い方法を利用して取引に関する決済を行うものとします。手数料やその他の取引関連費用についても、指定された方法で支払いを行います。

第8条（免責事項）

  1. サービスの提供

    2. 当サービスは、商品取引の仲介役に過ぎません。取引における問題（商品の不良、返却遅延、受け取り拒否等）について、当サービスは一切の責任を負いません。

    また、ユーザーが当サービスの利用で幸福になりすぎたり、笑いが止まらなくなった場合も、当サービスは責任を負いません。

  2. システム停止等

    3. 当サービスは、システムの保守、更新等によりサービスを一時的に停止することがあります。この場合、サービス停止によって生じた損害については責任を負いません。

    万一、当サービスが突然ログアウトした場合は、深呼吸して落ち着き、スマホを頭の上に乗せて「どうかログインできますように」と願ってください。それで何も変わらない場合、当サービスは責任を負いません。

第9条（個人情報の取り扱い）

当サービスは、ユーザーの個人情報を適切に取り扱い、プライバシーポリシーに従い保護します。ユーザーは、当サービスのプライバシーポリシーに同意のうえサービスを利用するものとします。

第10条（個人情報の第三者提供）

当社は，次に掲げる場合を除いて、あらかじめユーザーの同意を得ることなく、第三者に個人情報を提供することはありません。ただし、個人情報保護法その他の法令で認められる場合を除きます。

  1. ２人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき。
  2. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき。
  3. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき。
  4. 予め次の事項を告知あるいは公表し、かつ当社が個人情報保護委員会に届出をしたとき。
    1. 利用目的に第三者への提供を含むこと。
    2. 第三者に提供されるデータの項目。
    3. 第三者への提供の手段または方法。
    4. 本人の求めに応じて個人情報の第三者への提供を停止すること。
    5. 本人の求めを受け付ける方法。

第11条（年齢制限）

当サービスは、18歳以上のユーザーを対象としています。ユーザーは、当サービスを利用するために、登録時に自己の年齢が18歳以上であることを確認し、誓約するものとします。18歳未満のユーザーは、当サービスを利用することができません。

当サービスは、本規約を適宜変更することができ、変更後の規約は当サービス内に掲示された時点で効力を生じます。ユーザーは、定期的に規約の内容を確認する責任を負います。

第13条（未来予測）

当サービスには「未来予測機能」は一切ありません。ただし、当サービスを利用していると、気のせいで未来が明るくなることがあります。

  2. 本規約の変更が本サービス利用契約の目的に反せず、かつ、変更の必要性、変更後の内容の相当性その他の変更に係る事情に照らして合理的なものであるとき。

第17条（事故発生の対応手順）

報告対象事態が発生した場合は、以下のような観点を踏まえ必要な措置を講じる。

  1. 第一報

    2. 当社の従業員は、漏えい等の事案の発生を認識した場合には、個人データの取扱いに関する責任者（以下「責任者」という。）に報告をしなければならない。

  2. 事業者内部における報告及び被害の拡大防止

    3. 責任者は、前項の第一報があった場合、速やかに漏えい等の事案の防止その他の措置を講じるように関係部署に対して指示をする。

  3. 経営陣への報告

    4. 責任者は、必要と認められる場合、直ちに、代表取締役及び関係担当取締役に対して報告を行う。

  4. 事実関係の調査及び原因の究明

    5. 責任者は、関係部署と連携の上、以下の観点において漏えい等事案の事実関係の調査及び原因の究明に必要な措置を講ずる。

    • 漏えい等があった個人情報を取扱う部署及び担当者の特定
    • 漏えい等のルートの解明
    • 漏えい等の有無の確認（漏えい等していた場合には、漏えい先の特定を含む。）
    • 漏えい等の対象となる本人、情報の項目及び人数の特定
  5. 影響範囲の特定

    6. 責任者は、前条で把握した事実関係に関して、漏えい等の対象となる情報の本人の数、漏えいした情報の内容、発生した原因等を踏まえ、影響範囲を特定する。

  6. 再発防止策の検討及び実施

    7. 責任者は、本条第5項で究明した原因及び前項で特定した影響範囲を踏まえ、再発防止策を検討し、速やかに実施する。

  7. 個人情報保護委員会への報告及び本人への通知

    8. 責任者は、第5条及び第6条に従い、個人情報保護委員会への報告及び本人への通知を行う。

  8. 事実関係及び再発防止策の公表

    9. 責任者は、紛失等の事案が発生した場合、二次被害の防止、類似事案の発生回避等の観点から、事実関係及び再発防止策等について公表するか否か速やかに検討する。
